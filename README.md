# 💅 Miyottzi Moreno Nail Artist

## 🇪🇸 Español

Sitio web para un salón de uñas, creado como proyecto real para mostrar servicios y facilitar el contacto con clientes.

### ✨ Características

* Menú de servicios (manicure, pedicure, diseños, etc.)
* Enlace directo a Instagram
* Formulario de contacto
* Diseño limpio y responsivo

### 🚀 Tecnologías usadas

* HTML
* CSS (básico)
* Deploy en Netlify

### 🌐 Sitio Web en Vivo

👉 [https://miyottzimoreno.netlify.app/](https://miyottzimoreno.netlify.app/)

### 📌 Objetivo

Este proyecto fue desarrollado para el salón de uñas de mi esposa, con el objetivo de mostrar sus servicios y facilitar el contacto con clientes a través de internet.
